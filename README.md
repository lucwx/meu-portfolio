## Technologies Used
HTML, CSS, JavaScript, Vue3 (Vue Router), Tailwind CSS, NPM (Node Package Manager) and Vite.

## How To Run

### 1. Clone Repository
```sh
git clone https://github.com/lucwx/meu-portfolio.git
```

```sh
cd meu-portfolio
```

### 2. Install Dependencies

```sh
npm install
```

### 3. Run Project Locally

```sh
npm run dev
```
